# iaravida

Site do projeto iaravida — construído com Next.js, TypeScript e Tailwind, com deploy na Vercel.

## Rodar localmente

```bash
npm install
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).

A página principal está em `app/page.tsx` e atualiza automaticamente conforme você edita.

## Deploy

O site é implantado automaticamente na Vercel a partir do branch `main`.
